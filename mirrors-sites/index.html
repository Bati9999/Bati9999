国内开源镜像站汇总

By bati9999

2014-11-11 更新日期:2015-06-12

镜像站一般是为了解决速度问题，但是，在咱们这儿，没有镜像站的话，有些东西就如太监的下面—没有了，比如ruby gems（好像现在放在S3上），曾有段时间python也挂了！ 镜像站就是比驼峰航线还珍贵的生命线，在这里感谢那些为开源镜像站做出过贡献的人们。

下面的单子参考了网上各种“开源镜像站汇总” 、“开源镜像站大全”，去掉那些挂掉的（比如清华）、打开速度特别慢的（电信线路长城宽带）、项目特少的（如只有ubuntu或apache等）。我现在用阿里的镜像，还是可以的，偶尔有不畅。总的来说，下面的镜像站现在都是可以用的，企业的镜像站一般有各种linux和gems pypi等，几个大学的镜像站比企业的项目多，不仅有各种Linux，还有一些常用的开发工具，开源的语言等。

企业：

  • 阿里开源镜像站: http://mirrors.aliyun.com/
  • 搜狐开源镜像站： http://mirrors.sohu.com/
  • 网易开源镜像站： http://mirrors.163.com/
  • 淘宝ruby gems镜像: http://ruby.taobao.org/
  • 淘宝npm镜像: http://npm.taobao.org/
    用法：npm -g install package_name --registry=https://registry.npm.taobao.org
  • 豆瓣pypi镜像: http://pypi.douban.com/simple/
  • pypi官方的镜像列表：http://www.pypi-mirrors.org/用法：pip install package_name -i http://pypi.douban.com/simple
  • Composer 中国镜像: pkg.phpcomposer.com
  • PHP中国镜像: cn2.php.net|cn.php.net

大学:

  • 北京理工大学： http://mirror.bit.edu.cn (Mozilla, Mysql , PostgreSQL, Eclipse, VirtualBox)
  • 中国科学技术大学： http://mirrors.ustc.edu.cn
  • 北京交通大学： http://mirror.bjtu.edu.cn (Liberoffice)
  • 兰大开源社区镜像站: http://mirror.lzu.edu.cn
  • 浙江大学开源镜像站：http://mirrors.zju.edu.cn/ (PostgreSQL, Python(x,y) )
  • 清华大学 TUNA 镜像源: http://mirrors.tuna.tsinghua.edu.cn/ (Docker, homebrew )
